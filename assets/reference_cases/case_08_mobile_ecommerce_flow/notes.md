# Case 08 - Mobile E-commerce Flow Screens

## Source

Figma mobile wireframe reference, E-commerce page (`39:1116`).

## Useful Pattern

E-commerce screens are conversion flows. Product image, title, price, options, quantity, cart rows, summary, filter sheet, and primary checkout/add-to-cart CTA each have fixed semantic weight.

## Repair Intent

Protect product card grids, image-title-price pairing, sticky bottom CTAs, cart row alignment, filter sheet grouping, and checkout summary separation. Do not let promotional carousels overpower checkout actions.

## Common Failures

- Product grid gutters drift or cards resize inconsistently.
- Product image detaches from title/price.
- Add-to-cart or checkout CTA collides with the home indicator.
- Cart item quantity/price columns misalign.
- Filter controls overflow a bottom sheet.
