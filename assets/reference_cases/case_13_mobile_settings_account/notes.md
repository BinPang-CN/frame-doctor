# Case 13 - Mobile Settings/Account Screens

## Source

Figma mobile wireframe reference, Settings page (`0:1`).

## Useful Pattern

Settings screens are grouped row systems. Profile header, section groups, dividers, toggles, destructive actions, bottom tab bar, and modal settings panels must keep predictable structure.

## Repair Intent

Normalize row height, icon-label-chevron alignment, divider bounds, and group spacing. Preserve profile identity at the top and keep tab bars or modal panels separated from scrollable settings rows.

## Common Failures

- Row icons, labels, and chevrons lose a shared axis.
- Dividers extend beyond row group boundaries.
- Section headers float too far from their row group.
- Logout/destructive action looks like a normal row.
- Modal settings panel blends into the underlying screen.
